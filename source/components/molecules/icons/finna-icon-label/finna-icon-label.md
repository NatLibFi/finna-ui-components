### Description

An accessible icon and/or label.

### Properties

| Property           | Type         | Default Value |
|--------------------|--------------|---------------|
| `a11yLabel`        | array        |               |
| `class`            | array/string |               |
| `htmlLabel`        | boolean      | `false`       |
| `iconClass`        | array/string |               |
| `label`            | string       |               |
| `tooltip`          | string       |               |
| `tooltipPlacement` | string       | `top`         |

### Notes

`tooltipPlacement` can be `auto`, `top`, `right`, `bottom` or `left`.
